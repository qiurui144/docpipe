# docpipe v1.1 — API Hardening + End-to-End Ingest Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make docpipe's HTTP API end-to-end usable (upload → store → search) and stable: add `/v1/ingest`, document lifecycle management, a minimal async job model, and contract hardening, plus client + openapi updates.

**Architecture:** Extend `docpipe-core`'s `VectorStore` with a `documents` registry table (SqliteVecStore), have the facade `ingest` register documents and expose `ingest_file`/`list_documents`/`get_document`/`delete_document`. Add a deliberately-simple in-process async job queue in `docpipe-server` (in-memory map + a tokio semaphore — no Redis/heartbeat/TTL). Wire new axum routes (`ingest`, `documents`, `jobs`) with a body-size limit, update both clients with typed errors, and keep `openapi.yaml` aligned via a coverage test.

**Tech Stack:** Rust (edition 2021, MSRV 1.85), axum 0.8, rusqlite 0.32 + sqlite-vec, tokio (semaphore + spawn), async-trait. Python client: httpx + pydantic v2. TS client: fetch. Spec: `docs/superpowers/specs/2026-06-24-docpipe-v1.1-api.md`.

## Global Constraints

- **Backward compatible**: the existing 6 routes (`parse/chunk/embed/search/annotate/health`) and their response shapes do NOT change. New endpoints + additive fields only. `/v1` is additive-only; breaking changes would go to `/v2`.
- **docpipe-core stays clock-free**: core does NOT call `SystemTime::now`/`Date::now` (keeps it deterministic/testable). `created_at` (RFC3339) is **passed in by the caller** (the server generates it).
- **Async job model is deliberately simple** (per project rule "后台任务用简单串行队列，不引入 Redis/心跳/TTL/复杂调度"): an in-memory `Arc<Mutex<HashMap<String, Job>>>` + a `tokio::Semaphore(MAX_OCR_CONCURRENCY)`. Jobs are **ephemeral** (lost on restart) — documented, not persisted in v1.1.
- **Error codes are kebab-case** with fixed HTTP status (spec §7): `document-not-found`→404, `job-not-found`→404 (plus the existing 8). All errors flow through `DocError::code()`/`http_status()` → server `ApiError`.
- **chunk_id convention** is `"{doc_id}:{uuid}"`; document/vector deletion is by `doc_id` prefix; ingest is idempotent (delete-then-write; `documents` row upserted).
- **Comments in Chinese**; serde field names are snake_case (the HTTP/client contract). `cargo fmt` clean + `cargo clippy --workspace --all-targets -- -D warnings` clean after every task.
- **Repo**: `/data/company/project/docpipe`, branch `master`, commit per task, no push (push is a separate explicit step).

```rust
    async fn register_document(&self, info: &DocumentInfo) -> Result<()> {
        let conn = self.conn.lock().map_err(|_| DocError::VectorStoreError("lock".into()))?;
        conn.execute(
            "INSERT INTO documents (doc_id, collection, filename, format, page_count, chunk_count, created_at)
             VALUES (?1, ?2, ?3, ?4, ?5, ?6, ?7)
             ON CONFLICT(doc_id, collection) DO UPDATE SET
                filename=excluded.filename, format=excluded.format,
                page_count=excluded.page_count, chunk_count=excluded.chunk_count,
                created_at=excluded.created_at",
            rusqlite::params![
                info.doc_id, info.collection, info.filename, info.format,
                info.page_count, info.chunk_count, info.created_at
            ],
        )
        .map_err(|e| DocError::VectorStoreError(format!("register_document: {e}")))?;
        Ok(())
    }

    async fn list_documents(&self, collection: &str) -> Result<Vec<DocumentInfo>> {
        let conn = self.conn.lock().map_err(|_| DocError::VectorStoreError("lock".into()))?;
        let mut stmt = conn.prepare(
            "SELECT doc_id, collection, filename, format, page_count, chunk_count, created_at
             FROM documents WHERE collection=?1 ORDER BY created_at DESC",
        ).map_err(|e| DocError::VectorStoreError(format!("list prep: {e}")))?;
        let rows = stmt.query_map([collection], Self::row_to_doc)
            .map_err(|e| DocError::VectorStoreError(format!("list query: {e}")))?;
        let mut out = Vec::new();
        for r in rows { out.push(r.map_err(|e| DocError::VectorStoreError(format!("list row: {e}")))?); }
        Ok(out)
    }

    async fn get_document(&self, doc_id: &str, collection: &str) -> Result<Option<DocumentInfo>> {
        let conn = self.conn.lock().map_err(|_| DocError::VectorStoreError("lock".into()))?;
        let mut stmt = conn.prepare(
            "SELECT doc_id, collection, filename, format, page_count, chunk_count, created_at
             FROM documents WHERE doc_id=?1 AND collection=?2",
        ).map_err(|e| DocError::VectorStoreError(format!("get prep: {e}")))?;
        let mut rows = stmt.query_map(rusqlite::params![doc_id, collection], Self::row_to_doc)
            .map_err(|e| DocError::VectorStoreError(format!("get query: {e}")))?;
        match rows.next() {
            Some(r) => Ok(Some(r.map_err(|e| DocError::VectorStoreError(format!("get row: {e}")))?)),
            None => Ok(None),
        }
    }
```

```rust
//! 极简进程内异步任务队列 —— 内存注册表 + tokio Semaphore 限并发。
//! 无 Redis/心跳/TTL；job 重启即失（v1.1 ephemeral，spec §11）。

use std::collections::HashMap;
use std::future::Future;
use std::sync::{Arc, Mutex};

use docpipe_core::error::Result as CoreResult;
use docpipe_core::types::IngestResult;
use serde::Serialize;

#[derive(Debug, Clone, Serialize, PartialEq)]
#[serde(rename_all = "lowercase")]
pub enum JobStatus {
    Queued,
    Running,
    Done,
    Failed,
}

#[derive(Debug, Clone, Serialize)]
pub struct Job {
    pub job_id: String,
    pub status: JobStatus,
    pub created_at: String,
    pub result: Option<IngestResult>,
    pub error: Option<String>,
}

#[derive(Clone)]
pub struct JobQueue {
    jobs: Arc<Mutex<HashMap<String, Job>>>,
    sem: Arc<tokio::sync::Semaphore>,
}

impl JobQueue {
    pub fn new(max_concurrency: usize) -> Self {
        Self {
            jobs: Arc::new(Mutex::new(HashMap::new())),
            sem: Arc::new(tokio::sync::Semaphore::new(max_concurrency.max(1))),
        }
    }

    /// 提交一个产出 IngestResult 的 future，立即返回 job_id（状态 Queued）。
    pub fn submit<F>(&self, fut: F, created_at: String) -> String
    where
        F: Future<Output = CoreResult<IngestResult>> + Send + 'static,
    {
        let job_id = uuid::Uuid::new_v4().to_string();
        {
            let mut jobs = self.jobs.lock().unwrap();
            jobs.insert(job_id.clone(), Job {
                job_id: job_id.clone(), status: JobStatus::Queued,
                created_at, result: None, error: None,
            });
        }
        let jobs = self.jobs.clone();
        let sem = self.sem.clone();
        let id = job_id.clone();
        tokio::spawn(async move {
            let _permit = sem.acquire().await;
            if let Some(j) = jobs.lock().unwrap().get_mut(&id) { j.status = JobStatus::Running; }
            let outcome = fut.await;
            let mut g = jobs.lock().unwrap();
            if let Some(j) = g.get_mut(&id) {
                match outcome {
                    Ok(r) => { j.status = JobStatus::Done; j.result = Some(r); }
                    Err(e) => { j.status = JobStatus::Failed; j.error = Some(e.to_string()); }
                }
            }
        });
        job_id
    }

    pub fn get(&self, id: &str) -> Option<Job> {
        self.jobs.lock().unwrap().get(id).cloned()
    }
}
```

```rust
//! POST /v1/ingest —— multipart 文件一步入库；config.async=true 走异步 job。

use std::sync::Arc;

use axum::extract::{Multipart, State};
use axum::http::StatusCode;
use axum::Json;
use serde::Deserialize;

use crate::error::ApiError;
use crate::state::AppState;
use docpipe_core::error::DocError;
use docpipe_core::types::ParseConfig;

#[derive(Deserialize, Default)]
pub struct IngestConfig {
    #[serde(default)] pub ocr: Option<bool>,
    #[serde(default)] pub table_structure: Option<bool>,
    #[serde(default)] pub max_pages: Option<u32>,
    #[serde(default)] pub dpi: Option<u32>,
    #[serde(default)] pub r#async: bool,
    #[serde(default = "default_collection")] pub collection: String,
}
fn default_collection() -> String { "default".into() }

pub async fn ingest(State(state): State<Arc<AppState>>, mut mp: Multipart)
    -> Result<(StatusCode, Json<serde_json::Value>), ApiError> {
    let mut bytes: Option<Vec<u8>> = None;
    let mut filename: Option<String> = None;
    let mut cfg = IngestConfig::default();
    cfg.collection = default_collection();
    while let Some(field) = mp.next_field().await
        .map_err(|e| ApiError(DocError::Other(format!("multipart: {e}"))))? {
        match field.name() {
            Some("file") => {
                filename = field.file_name().map(|s| s.to_string());
                bytes = Some(field.bytes().await
                    .map_err(|e| ApiError(DocError::Other(format!("file read: {e}"))))?.to_vec());
            }
            Some("config") => {
                let txt = field.text().await
                    .map_err(|e| ApiError(DocError::Other(format!("config read: {e}"))))?;
                if !txt.trim().is_empty() {
                    cfg = serde_json::from_str(&txt)
                        .map_err(|e| ApiError(DocError::Other(format!("config json: {e}"))))?;
                }
            }
            _ => {}
        }
    }
    let bytes = bytes.ok_or_else(|| ApiError(DocError::Other("missing file field".into())))?;
    let parse_cfg = ParseConfig {
        ocr: cfg.ocr.unwrap_or(true),
        table_structure: cfg.table_structure.unwrap_or(false),
        max_pages: cfg.max_pages,
        dpi: cfg.dpi.unwrap_or(300),
    };
    let created_at = chrono::Utc::now().to_rfc3339();
    let collection = cfg.collection.clone();
    if cfg.r#async {
        // sdk 是 Clone? Docpipe 内部是 Arc 字段 → 用 state.clone() 持有 Arc<AppState> 进 future。
        let state2 = state.clone();
        let fname = filename.clone();
        let ca = created_at.clone();
        let job_id = state.jobs.submit(async move {
            state2.sdk.ingest_file(&bytes, fname.as_deref(), parse_cfg, &collection, &ca).await
        }, created_at);
        Ok((StatusCode::ACCEPTED, Json(serde_json::json!({ "job_id": job_id, "status": "queued" }))))
    } else {
        let r = state.sdk.ingest_file(&bytes, filename.as_deref(), parse_cfg, &collection, &created_at).await?;
        Ok((StatusCode::OK, Json(serde_json::to_value(r).unwrap())))
    }
}
```

> **`ingest_file` in a spawned future**: `Docpipe` must be reachable from `'static` Send future. `AppState` is held as `Arc<AppState>`; the async route captures `state.clone()` (an `Arc`), so the future owns an `Arc<AppState>` → `Send + 'static`. (`Docpipe`'s fields are all `Arc<dyn ... + Send + Sync>`, so this holds.) If the borrow checker complains about borrowing `bytes`, move it into the async block (it already is via `async move`).

```python
import json as _json, time as _time

def _check(self, r):
    if r.status_code >= 400:
        try:
            body = r.json(); code = body.get("error", f"http-{r.status_code}"); detail = body.get("detail", "")
        except Exception:
            code = f"http-{r.status_code}"; detail = r.text
        raise DocpipeError(code, detail, r.status_code)
    return r

def ingest_bytes(self, data: bytes, filename: str = "file", collection: str = "default", async_: bool = False, **config) -> dict:
    cfg = {**config, "collection": collection, "async": async_}
    r = self._check(self._client.post(f"{self.base_url}/v1/ingest",
        files={"file": (filename, data)}, data={"config": _json.dumps(cfg)}))
    return r.json()

def ingest_file(self, path: str, collection: str = "default", async_: bool = False, **config) -> dict:
    with open(path, "rb") as f:
        return self.ingest_bytes(f.read(), path.split("/")[-1], collection, async_, **config)

def list_documents(self, collection: str = "default") -> list:
    return self._check(self._client.get(f"{self.base_url}/v1/documents", params={"collection": collection})).json()["documents"]

def get_document(self, doc_id: str, collection: str = "default") -> dict:
    return self._check(self._client.get(f"{self.base_url}/v1/documents/{doc_id}", params={"collection": collection})).json()

def delete_document(self, doc_id: str, collection: str = "default") -> dict:
    return self._check(self._client.delete(f"{self.base_url}/v1/documents/{doc_id}", params={"collection": collection})).json()

def get_job(self, job_id: str) -> dict:
    return self._check(self._client.get(f"{self.base_url}/v1/jobs/{job_id}")).json()

def wait_for_job(self, job_id: str, timeout: float = 300, poll: float = 1.0) -> dict:
    deadline = _time.time() + timeout
    while _time.time() < deadline:
        j = self.get_job(job_id)
        if j["status"] in ("done", "failed"):
            return j
        _time.sleep(poll)
    raise DocpipeError("job-timeout", f"job {job_id} not done in {timeout}s")
```

```typescript
async ingestFile(file: Blob | Uint8Array, filename: string, opts: { collection?: string; async?: boolean; ocr?: boolean } = {}): Promise<IngestResult> {
  const form = new FormData();
  const blob = file instanceof Blob ? file : new Blob([file]);
  form.append("file", blob, filename);
  form.append("config", JSON.stringify({ collection: opts.collection ?? "default", async: opts.async ?? false, ocr: opts.ocr }));

async waitForJob(jobId: string, opts: { timeoutMs?: number; pollMs?: number } = {}): Promise<Job> {
  const deadline = Date.now() + (opts.timeoutMs ?? 300000);
  while (Date.now() < deadline) {
    const j = await this.getJob(jobId);
    if (j.status === "done" || j.status === "failed") return j;
    await new Promise((res) => setTimeout(res, opts.pollMs ?? 1000));
  }
  throw new DocpipeError("job-timeout", jobId, 0);
```
